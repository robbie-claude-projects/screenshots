Selector format examples:
  • #ad-container - Element with ID "ad-container"
  • .advertisement - Elements with class "advertisement"
  • [data-ad-slot] - Elements with data-ad-slot attribute
  • iframe[src*="ads"] - Iframes with "ads" in src
